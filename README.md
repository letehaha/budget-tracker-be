# budget-tracker-be
Budget tracker back-end

### Run instruction

1. Install dependencies
```sh
npm i
```

2. Fill config with required data (like DB name, user, pass, and e.g.) for dev/prod environments

3. Run migrations
```sh
npm run migrations
```

4. Run seeds

```sh
npm run seed
```

5. Start dev server

```sh
npm run dev
```
